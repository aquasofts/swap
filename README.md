<h1>为小白提供一键开启swap脚本</h1>
运行本脚本将会自动开启3GB的swap空间，并设置开机自动开启。

### 使用方法
一键脚本：
`wget -N --no-check-certificate https://raw.githubusercontent.com/aquasofts/swap-/main/swap.sh && chmod +x swap.sh && ./swap.sh`
